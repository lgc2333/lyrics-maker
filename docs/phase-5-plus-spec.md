# Phase 5 Plus Spec

## Bug

### 无音频状态 / 边界行为

- 在载入工程但是未导入音频的情况下，限制用户能进行的操作
- 在没导入音频的状态下，要能自由切换频谱与歌词模式，注意处理编辑模式下的快捷键与操作，在没导入音频时的边界状态下的行为（一般是忽略）

### 时间线 / 滚动跟随

- MainView的播放指针在MainView中间左侧的位置时，也不要自动居中，等到跑到中间了再继续自动居中
- 拉动进度条时，自动让MainView跟踪到进度条位置
- 点击歌词栏的歌词后，自动跳转进度时，MainView应跟随滚动到位

### 时间线 / Overlay 显示

- 目前时轴上不显示未打轴完全的歌词行，需要让它显示（需要处理边界情况，比如只显示拥有startTime的歌词行，且只遍历显示数组顺序中依次拥有endTime的词，一旦遇到没有endTime的词则结束渲染，不显示结尾的颜色线，结尾只显示分词虚线）

### 打轴 / 网格设置 / 快捷键

- 吸附按钮目前仍在禁用状态，使其可以使用
- 打轴按Enter键进入下一句时，当前词位置应永远处在起始处

### 菜单栏 / 撤销重做

- 使编辑菜单中的撤销/重做按钮可用，并显示当前操作的名称

## Adjust

### 时间线 / Overlay 样式

- LineOverlay用于分词的虚线可读性很差，换成黄色

### 频谱 / 缩放行为

- 频谱缩放应以鼠标位置为中心

### 音量 / 本地存储

- 音量的存储应存放在LocalStorage而不是项目中

### 网格设置 / 控件样式

- 将普通、三连音切换的下拉框变成三态按钮（普通、三连音、Alt触发的三连音）
- 将分隔倍数切换的下拉框变成左-右+中数值的样式

### 切词栏 / 时轴模式

- 切词栏时轴模式，起始结束时间放在栏最右侧，保持单行style，样式修改为
  - 起始块：（输入框，时分秒格式，按回车应用）
  - 其他块：xx:xx.xxx ~ （如上输入框）

### 时轴列表 / 控件样式

- 时轴列表的删除键改为图标

### 菜单栏 / 标题栏

- 使菜单栏弹出的菜单能够自动切换到hover的顶级菜单项，如果有二级菜单也可以自动hover展开
- 重新组织菜单项：
  - 文件：新建工程、打开工程、打开音乐、保存项目、项目另存为、首选项
  - 编辑：撤销、重做
  - 帮助：关于

### 本地化

- Offset、Set offset to current time 按钮、Tap to get BPM按钮的文本需要本地化

## New Feature

### 项目 / 保存 / 状态提示

- 在界面最底部放置StatusBar，最左边显示提示信息（保存，撤销，重做状态，各种提示等等）
- 项目工程保存（快捷键Ctrl+S）
- 项目每分钟自动保存（在有拿到FileHandle的情况下才成功触发）
- 项目工程实时自动保存到浏览器本地，下次打开自动读取（读取之后标题栏显示的保存状态须为未保存）
- 标题栏中的标题应显示 \*(当前工程修改状态)工程名字；点击这里变成编辑框直接修改工程名字

### 导入导出 / 文件交互

- TXT/LRC/ASS 可方便扩展的导入导出架构
- 导入操作可以拖动文件进来，然后加一个popup modal提示框确认是否导入

### 本地设置 / 首选项

- 快捷键自定义 UI，定义的快捷键保存到浏览器本地
- 浏览器本地设置导入导出

### 时间线 / 视图 / Overlay

- 让用户可以在LineOverlay上拖动每行歌词的起始结束点以及每个词的起始结束点
- 在GridOverlay上添加当前鼠标指针位置的时间预览
- 在TransportBar上添加一个不让当前MainView播放进度指针自动居中的开关

### 音频 / 播放 / 音量

- 歌曲支持 25%/50%/75%/100% 切换变速，可以做在音量按钮左边
- 音量/音效按钮支持点击静音/解除静音

### 打轴 / 网格设置 / 快捷键

- 用于打轴的D键应可以绑定一个副键位如S，高速歌词可以分担手指压力

### 歌词编辑 / 歌词列表

- 歌词模式下支持Ctrl+V导入剪贴板中的歌词文本
- 歌词列表每列右侧按钮快捷操作（全部使用图标）
  - 可以删除歌词行
  - 可以在该歌词行上下添加一行空行
  - 可以重新排序歌词行
